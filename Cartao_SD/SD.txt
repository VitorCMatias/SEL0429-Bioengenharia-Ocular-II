# Cartão SD

Este módulo é responsável por fazer tanto a leitura quanto a escrita de dados no cartão SD da equipe, para o desenvolvimento desse código foram utilizados os materiais listados a baixo, tanto o vídeo do Youtube, quanto a webpage são da mesma pessoa e uma funciona como complemento para a outra.

- [Vídeo do Youtube](https://www.youtube.com/watch?v=aqSNz26Cuio), o vídeo não é para o sensor que utilizamos, mas a escrita do arquivo foi feita corretamente.
- [Webpage](https://www.micropeta.com/video29), o material desse site complementa o do vídeo, além disso nele pode serm ser encontradas as bibliotecas auxiliares, bem como os locais do código que devem ser alterados.
